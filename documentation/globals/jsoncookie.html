JSONCookie - Global Method

Summary

Represents a cookie with a value that's serialized as JSON..

Syntax

JSONCookie(String, Object, DateTime/TimeSpan)

Parameters

name

Type: String
Required
The name of the cookie.value

Type: Object
Required
(Optional)The value of the cookie.time

Type: DateTime/TimeSpan
Required
(Optional)If a DateTime, this sets the expiration date of the cookie. If a TimeSpan, DateTime.Now() adds the TimeSpan for the expiration date.

Type:
