ing.Format - Global Method

Summary

Replaces the given place holders indicated with {#} where # is the index number of the the provided parameters.

Syntax

ing.Format(String, Variadic)

Parameters

string

Type: String
Required
String with placeholders.variadicParams

Type: Variadic
Required
(Variadic) Any number of values that corresponds with number of placeholders.

Return Type

Type: String
Returns a copy of the instance with placeholders replaced with values from the provided parmeters.
