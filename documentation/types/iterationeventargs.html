IterationEventArgs - Object Type

Summary

Contains information about a array iteration event from calling ForEach or ForEachReversed.

Constructor

new IterationEventArgs(Object sender, Object data)

Contains information about a array iteration event from calling ForEach or ForEachReversed.

NameTypeRequiredDefaultSummary
senderObjectyesundefinedThe object sending the event.
dataObjectyesundefined(Optional)Any data to send to the handlers.

Fields

NameTypeRequiredDefaultSummary
CancelEventBooleanIf set to true, ForEach loop stops.
IndexNumberCurrent index of the iteration.
ObjectObjectCurrent object of the array.
SenderObjectThe object that sent the event.
NavigateNumberUsed to control looping. Changing this will affect the next iteration. A postive number means skipping the number of elements. A negative number means going back to already touched elements. After each iteration, Navigate is set back to 0.
