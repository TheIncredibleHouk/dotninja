Timer - Object Type

Summary

Creates a timer that fires a trigger after each period.

Constructor

new Timer(Number/TimeSpan millisecondsTimeSpan, Boolean loop)

Creates a timer that fires a trigger after each period.

NameTypeRequiredDefaultSummary
millisecondsTimeSpanNumber/TimeSpanyesundefinedIf a number, treated as total milliseconds that each period lasts. If a TimeSpan, the amount of time for a period to last.
loopBooleanyesundefinedIf true, once the period happens, the timer will execute again.

Fields

NameTypeRequiredDefaultSummary
LoopBooleanIf set to true, the timer will continue to run. If set to false, the timer will stop running after the next trigger.
RunningBooleanTells if the timer is currently active.
TagObjectAny object for the timer to hold.
TypeStringReturns Type.Timer.
