Delegate - Object Type

Summary

A specialized function that calls the provided function with the provided caller and with the provided parameters passed in.

Constructor

new Delegate(Function func, Object caller, Array parameters)

A specialized function that calls the provided function with the provided caller and with the provided parameters passed in.
